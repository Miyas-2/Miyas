# Portofolio Repositori GitHub - Moh Ilyas

Dokumentasi seluruh proyek, tugas kuliah, dan eksperimen pemrograman yang dikembangkan oleh Moh Ilyas (`Miyas-2`), Mahasiswa Informatika di Institut Teknologi Nasional (Itenas) Bandung.

## 🚀 Artificial Intelligence & Data Science

### [Kayuin](https://github.com/Miyas-2/Kayuin)

- **Deskripsi:** Proyek eksperimen yang mengintegrasikan fitur _Speech Recognition_ (pengenalan suara) dan _Text to Speech_ (mengubah teks menjadi suara).
- **Tech Stack:** Jupyter Notebook, Python.

### [Data-Retrival-Enhanced-Confix-Stripping-TF-IDF-Cosine-Similarity](https://github.com/Miyas-2/Data-Retrival-Enhanced-Confix-Stripping-TF-IDF-Cosine-Similarity)

- **Deskripsi:** Sistem Temu Kembali Informasi (_Information Retrieval_) menggunakan algoritma _Confix Stripping_ untuk proses _stemming_ teks bahasa Indonesia, pembobotan TF-IDF, dan pengukuran tingkat kemiripan dokumen menggunakan _Cosine Similarity_.
- **Tech Stack:** Jupyter Notebook, Python.

### [Deteksi-Nominal-IDR](https://github.com/Miyas-2/Deteksi-Nominal-IDR)

- **Deskripsi:** Proyek berbasis _computer vision_ untuk mendeteksi dan mengenali nilai nominal pada mata uang Rupiah (IDR).
- **Tech Stack:** Python.

### [Parallel-vs-vanilla-vs-vektorisasi](https://github.com/Miyas-2/Parallel-vs-vanilla-vs-vektorisasi)

- **Deskripsi:** Proyek komparasi performa komputasi untuk mengukur perbedaan kecepatan eksekusi kode antara metode paralel, _vanilla_ (looping biasa), dan teknik vektorisasi.
- **Tech Stack:** Jupyter Notebook, Python.

### [Tumbuhkan_Streamlit](https://github.com/Miyas-2/Tumbuhkan_Streamlit)

- **Deskripsi:** Antarmuka pengguna (_frontend_) dari platform/aplikasi "Tumbuhkan" yang diimplementasikan menggunakan framework Streamlit agar model data dapat diakses via web.
- **Tech Stack:** Jupyter Notebook, Streamlit.

### [Backend_Tumbuhkan](https://github.com/Miyas-2/Backend_Tumbuhkan)

- **Deskripsi:** Sisi server (_backend_) yang menangani logika bisnis dan pemrosesan data utama untuk menyokong aplikasi "Tumbuhkan".
- **Tech Stack:** Python.

### [Big-Data-Final-Exam](https://github.com/Miyas-2/Big-Data-Final-Exam)

- **Deskripsi:** Repositori pengerjaan ujian akhir semester (UAS) yang berfokus pada analisis dan pengolahan data skala besar (_Big Data_).
- **Tech Stack:** Jupyter Notebook, Python.
